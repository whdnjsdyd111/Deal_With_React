이 장에서는 Node.js 서버와 MongoDB를 연동할 수 있도록 MongoDB 기초 지식을 알아본다.
그리고 mongoose를 이용하여 서버에서 직접 데이터를 추가 조회 삭제 수정 하는 방법도 알아본다.
이번 실습은 다음 흐름으로 진행된다.


    MongoDB 기본 지식
        ↓
    작업 환경 설정
        ↓
    mongoose로 데이터베이스 연결
        ↓
    esm으로 ES 모듈 import/export 문법 사용
        ↓
    스키마와 모델 이해
        ↓
    REST API 구현
        ↓
    페이지네이션 구현
